# 나의 역할
1. 서버
> - Apache2 웹서버 설계 및 구축
> - 백업 보관소 용도의 Azure클라우드와 가상머신 구축
2. 소프트웨어
> - 소프트웨어 설계와 구축, 웹앱 개발
3. PM
> - 계획 일정 정리 및 문서화 작업
> - 전체적인 기획 총괄
4. FE
> - Laravel Frame Work
> - Grafana 대시보드 Embed
5. BE
> - Post 통신 버튼 구현
> - KakaoTalk Message API 활용
